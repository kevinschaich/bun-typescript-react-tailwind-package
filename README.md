# bun-typescript-react-tailwind-package

Features:

* [Bun](https://bun.sh/)
* [TypeScript](https://www.typescriptlang.org/)
* [React](https://react.dev/)
* [Tailwind](https://tailwindcss.com/)
* [Prettier]()
* Prettier tailwind
* Prettier imports
* GH Actions

Install:

```bash
bun install
```

Build:

```bash
bun run
```
